# List Decoding of Logarithmic Codes: Verification

Код воспроизводит вычислительную проверку списочного декодера в смешанной
метрике Ли — Хэмминга.

## Запуск

Требуется Python 3.9 или новее. Сторонние зависимости отсутствуют.

```bash
python3 verify_article.py
```

## Проверки

- матрица и два примера кода над `GF(11)`;
- полный перебор кода над `GF(5)`: 768 сравнений и распределение размеров
  списков;
- 4835 шаблонов смешанных ошибок радиуса 4 над `GF(11)`;
- 200 сравнений декодера с независимым локальным перебором.

Успешный запуск завершается сообщением:

```text
complete weighted list decoder verified
```

GitHub Actions автоматически выполняет ту же проверку для Python 3.9 и 3.12.
